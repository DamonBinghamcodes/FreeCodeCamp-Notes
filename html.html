HTML Notes - Beginner to Advanced

What is HTML?

HTML (HyperText Markup Language) is the backbone of every website. It tells the browser what to display — like text, images, links, forms, etc.

It uses "tags" (like <p> and <img>) to define elements on the page.

Real-World Use: HTML is used to build the structure of every web page — from basic landing pages to complex web apps.

HTML Document Structure

<!DOCTYPE html>  <!-- Tells the browser this is an HTML5 document -->
<html lang="en">
  <head>
    <meta charset="UTF-8"> <!-- Character encoding -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0"> <!-- Responsive design -->
    <title>Page Title</title> <!-- Tab title -->
  </head>
  <body>
    <!-- Content goes here -->
  </body>
</html>

Common Tags

  • <h1> to <h6> – Headings (used for titles and hierarchy)
  • <p> – Paragraph
  • <a href="url"> – Link to another page or site
  • <img src="image.jpg" alt="description"> – Show images
  • <br> – Line break
  • <strong> and <em> – Bold and italic text (for emphasis)
  • <ul>, <ol>, <li> – Lists (unordered and ordered)

Use Case: Use these tags to structure any webpage's main content, from blog posts to product descriptions.

Forms (User Input)

Used to collect user data like emails, feedback, search queries, etc.

<form action="/submit" method="POST">
  <label>Name: <input type="text" name="name"></label>
  <input type="submit" value="Send">
</form>

Use Case: Forms power login systems, contact pages, checkout screens, etc.

Semantic HTML

Tags that give meaning to the content: <header>, <nav>, <main>, <section>, <footer>

They improve SEO and accessibility.

Use Case: Essential for clean, maintainable code in real-world apps and professional-grade sites.

Advanced Features

  • <video>, <audio> – Embed media
  • <canvas> – For drawing/animations with JavaScript
  • <iframe> – Embed another webpage
  • <meta> – For SEO, responsive design, sharing info
  • <data> and data-attributes – Store custom data for scripts

Use Case: Used in interactive apps, games, social media embeds, analytics, and dynamic content.

Pro Tips

  • Use semantic tags for SEO & clarity
  • Always use alt text on images for accessibility
  • Structure content with headings logically (only one <h1> per page!)
  • Validate your HTML with W3C Validator
  • Use browser DevTools (Right-click → Inspect)
