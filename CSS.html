CSS Notes - Beginner to Advanced

What is CSS?

CSS (Cascading Style Sheets) controls how HTML elements look on the page. It defines layout, colors, fonts, spacing, animations, and much more.

Why it matters: Without CSS, all websites would look like plain text documents. CSS creates the design and visual experience.

How CSS Works

  • Selectors: Target which HTML elements you want to style.
  • Properties: Define what style to apply (like color, font-size).
  • Values: Assign the desired value to that property.
p {
  color: red;
}

Explanation: This sets all <p> elements to display red text.

Ways to Use CSS

  • Inline: Style added directly in the tag using the "style" attribute. Bad practice for big projects.
  • Internal: CSS inside a <style> tag in the HTML head. OK for small pages.
  • External: Best practice. Link to a .css file using <link> in your HTML head.

Box Model

Everything in CSS is a box.

  • Content: The actual text/image inside
  • Padding: Space between content and border
  • Border: Edge around the element
  • Margin: Space outside the element (between elements)

Why it matters: Understanding box model is key to controlling layout and spacing.

Positioning

  • Static: Default. Elements follow normal flow.
  • Relative: Move based on normal position.
  • Absolute: Position relative to closest parent with position set.
  • Fixed: Fixed to the screen, doesn't move.
  • Sticky: Sticks to screen after scroll starts.

Flexbox

Powerful layout system for 1D (rows or columns).

display: flex;
justify-content: center;
align-items: center;

Use Case: Center elements, create navbars, responsive layouts.

Grid

Layout system for 2D (rows and columns).

display: grid;
grid-template-columns: repeat(3, 1fr);

Use Case: Design complex layouts like dashboards, image galleries.

Media Queries (Responsive Design)

@media (max-width: 768px) {
  body {
    font-size: 0.9rem;
  }
}

Why: CSS that adapts to different screen sizes (mobile, tablet, desktop).
